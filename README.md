## :car: Getting Started

### :gear: Installation

1. Clone my Repo
```bash
  git clone https://github.com/BernaDeryaDen/InstagramCloneApp.git

```
2. Install Expo globally

```bash
npm install --global expo
```

3. Enter into the Project

```bash
   cd Instagram-bdd-clone-main
```

4. Install all dependencies

```bash
   npx expo install
```

5. Start the server

```bash
   expo start
```
